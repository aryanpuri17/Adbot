# 💰 CryptoTaskBot — Setup Railway

## Variables d'environnement à configurer sur Railway

Va dans ton projet Railway → Variables → Ajoute ces variables :

| Variable | Valeur | Obligatoire |
|---|---|---|
| `BOT_TOKEN` | Token de ton bot BotFather | ✅ Oui |
| `ADMIN_IDS` | Ton ID Telegram (ex: 6339278677) | ✅ Oui |
| `WALLET_TON` | Ton adresse TON | ✅ Oui |
| `WALLET_USDT_TRC20` | Ton adresse USDT TRC20 | ⚠️ Si activé |
| `WALLET_USDT_BEP20` | Ton adresse USDT BEP20 | ⚠️ Si activé |
| `PORT` | 3000 | ✅ Oui |
